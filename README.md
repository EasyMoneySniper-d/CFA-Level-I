大家好，这个Program主要是给大家分享我备考CFA-Level-I的一些经历和学习经验。本人存在一些经济学/数理基础，所以备考周期大概在4-5个月左右。其中前三个月是系统性的学习了CFA的课程，了解了CFA的知识体系，后一个半月是用来刷题和巩固错题。
我完成了官网上的5个Mock和1000题的pp（刷了两遍），先写的PP，正确率在一刷时期的63%（以财报举例），到二刷的82%。我认为pp的正确率要稳定在80%以上就对于Pass考试就非常稳了，但我的目标是10A，所以我的一些经历不适用于所有人，但照着我的笔记和我的思路来学习的话，Pass基本上是十拿九稳。

CFA主要是培养投资经理人为目的，并不具有普适性，我认为从事买方/资管用处会大一些。CFA可能更偏向培养一个你自己投资思维的框架，会对未来身价不菲的各位做资产配置时，提供一定程度的知识积累和专业判断；这个考试并不能说对你求职以及工作有质的改变，但你备考时的坚持，对于知识系统性框架的建立，对于职业道德的遵守和规范，都会在未来的职场和人生中发挥重要的作用。

CFA有十个科目，个人认为Level-I最难的应该是财务（FSA）和固收（Fixed Income），衍生品（Derivative）虽然是公认最难的科目，但因占比很小且学的很浅，我认为只要愿意学就是可以学到A的。

CFA为全英文考试，故我的笔记以英文为主，但我搭配了大量中文注解来辅助理解，并且紫色和红色颜色笔大多数都标记为考点，需着重复习。
